◆ coeff_modulus_128()

static std::vector<SmallModulus> seal::DefaultParams::coeff_modulus_128 ( std::size_t  poly_modulus_degree)
inlinestatic

Returns the default coefficients modulus for a given polynomial modulus degree. The polynomial modulus and the coefficient modulus obtained in this way should provide approdimately 128 bits of security against the best known attacks, assuming the standard deviation of the noise distribution is left to its default value.

Parameters
[in]poly_modulus_degreeThe degree of the polynomial modulus
Exceptions
std::out_of_rangeif poly_modulus_degree is not 1024, 2048, 4096, 8192, 16384, or 32768
