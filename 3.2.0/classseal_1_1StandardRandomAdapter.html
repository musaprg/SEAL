Detailed Description

template<typename RNG>
class seal::StandardRandomAdapter< RNG >

Provides an implementation of UniformRandomGenerator for the standard C++ library's uniform random number generators.

Template Parameters
RNGspecifies the type of the standard C++ library's random number generator (e.g., std::default_random_engine)
