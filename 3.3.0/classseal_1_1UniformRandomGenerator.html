Detailed Description

Provides the base class for a uniform random number generator. Instances of this class are typically returned from the UniformRandomGeneratorFactory class. This class is meant for users to sub-class to implement their own random number generators. The implementation should provide a uniform random unsigned 32-bit value for each call to generate(). Note that the library will never make concurrent calls to generate() to the same instance (but individual instances of the same class may have concurrent calls). The uniformity and unpredictability of the numbers generated is essential for making a secure cryptographic system.

See also
UniformRandomGeneratorFactory for the base class of a factory class that generates UniformRandomGenerator instances.
