Detailed Description

Class to store a secret key.

Thread Safety
In general, reading from SecretKey is thread-safe as long as no other thread is concurrently mutating it. This is due to the underlying data structure storing the secret key not being thread-safe.
See also
KeyGenerator for the class that generates the secret key.
PublicKey for the class that stores the public key.
RelinKeys for the class that stores the relinearization keys.
GaloisKeys for the class that stores the Galois keys.
